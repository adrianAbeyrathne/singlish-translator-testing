# Singlish to Sinhala Translator – Playwright Testing Project

## Module
ITPM / Software Quality Assurance

## Project Description
This project contains automated functional and UI test cases developed using the
Playwright testing framework to evaluate a Singlish to Sinhala language translation system.

The test suite includes:
- Positive functional test cases
- Negative and failure scenarios
- Sentence structure variations
- Informal and formal language handling
- Mixed English–Singlish inputs
- Typographical and robustness testing
- Real-time UI translation behavior

---

## Technologies Used
- Node.js
- Playwright Test Framework
- JavaScript
